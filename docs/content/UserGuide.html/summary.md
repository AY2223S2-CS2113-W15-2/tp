## Command Summary

| Command | Format                                            |
|--------:|---------------------------------------------------|
|    /add | `/add DESCRIPTION -c CATEGORY -p PRICE`          |
|   /view | `/view ENTRY_ID -c CATEGORY`                      |
|   /edit | `/edit ENTRY_ID -d DESCRIPTION -c CATEGORY -p PRICE` |
| /delete | `/delete ENTRY_ID`                                |
|   /help | `/help`                                           |
